Landing page sofisticada para serviços de bar e coquetelaria premium da BA Eventos.

## Última Atualização

## Tecnologias
- React
- Vite
- Tailwind CSS
- Lucide React

## Como executar
1. Instale as dependências: `npm install`
2. Inicie o servidor de desenvolvimento: `npm run dev`
